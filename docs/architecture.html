ZombieCoder: আর্কিটেকচার ও ব্যবস্থাপনা

  • Core: Python (Dispatcher, fallback, latency log)
  • Laravel API: Provider sync, Admin UI, DB management
  • Vue/React UI: Admin panel, fallback config, live status
  • Providers: YAML config (cloud/local), fallback-aware
  • Agents: YAML/DB—personality, fallback, memory
  • Cache: Per-provider/model, smart fallback-aware
  • Logs: latency, fallback, agent status

প্রতিটি টেকনোলজি বেছে নেওয়া হয়েছে—স্কেল, কাস্টমাইজেশন, এবং ওপেন সোর্স ফ্লেক্সিবিলিটির জন্য।
