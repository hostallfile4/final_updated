ZombieCoder: Android Studio ইন্টিগ্রেশন

  1. ZombieCoder server চালু করুন (local বা cloud)
  2. Android Studio-তে custom Gradle task যোগ করুন:
    task connectToMCP(type: Exec) { commandLine 'curl', '-X', 'POST', 'http://localhost:8000/instruct', '-d', 'query=build me a login activity' }
  3. কমান্ড চালান: ./gradlew connectToMCP
  4. Response পাবেন: AI-generated code, doc, বা suggestion
  5. API key থাকলে header-এ দিন

এভাবে Android Studio থেকে সরাসরি ZombieCoder-এর AI এজেন্ট, কোড, ডক, fallback—all ব্যবহার করতে পারবেন।
