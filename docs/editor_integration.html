ZombieCoder: এডিটর ইন্টিগ্রেশন (VS Code & Cursor)

ZombieCoder-এর AI suggestion, codegen, docgen, ও fallback ফিচার VS Code, Cursor, বা অন্য কোড এডিটরে ব্যবহার করতে নিচের ধাপগুলো অনুসরণ করুন:

১. Local Endpoint কনফিগারেশন

  1. Admin UI বা API থেকে আপনার local endpoint (যেমন: http://localhost:9000/v1/completions) কপি করুন।
  2. এন্ডপয়েন্টটি VS Code বা Cursor-এর settings.json-এ যুক্ত করুন।

২. Cursor Editor Integration

Cursor-এর জন্য settings.json ফাইলে নিচের কনফিগারেশন দিন:

{
  "ai.agent": {
    "endpoint": "http://localhost:9000/v1/completions",
    "localModels": {
      "codeAnalysis": "mistral",
      "codeGeneration": "deepseek",
      "documentation": "gemma"
    },
    "fallbackToLocal": true
  }
}

৩. VS Code Integration

VS Code-এর জন্য settings.json ফাইলে দিন (LocalAI/CodeGPT extension):

{
  "localAI.enable": true,
  "localAI.endpoint": "http://127.0.0.1:9000",
  "localAI.modelMapping": {
    "default": "mistral",
    "documentation": "gemma",
    "codegen": "deepseek"
  }
}

৪. Model Mapping (YAML/JSON)

প্রয়োজনে YAML/JSON ফাইলে model mapping কনফিগার করুন:

model_mapping:
  code: mistral
  doc: gemma
  chat: deepseek

৫. Fallback Chain ও API Key

  • Fallback chain ও API key এখন OpenAI, TogetherAI, Ollama, HuggingFace, lmstudio—সব provider 100% enabled।
  • API key না থাকলে cloud provider কাজ করবে না।

৬. Troubleshooting

  • Endpoint connection error দেখলে: server/app.py বা dispatcher/core.py চালু আছে কিনা চেক করুন।
  • Model না দেখালে: curl http://localhost:9000/status দিয়ে চেক করুন।
  • VS Code/ Cursor extension restart দিন প্রয়োজনে।

৭. ডকুমেন্টেশন ও হেল্প
