ZombieCoder: API ব্যবহার

ZombieCoder API ব্যবহার করতে:

  1. API endpoint: POST /agent/run (Laravel/Flask)
  2. Request body (JSON):
    { "agent": "instruct", "input": "বাংলায় একটা গল্প লেখো" }
  3. Response:
    { "reply": "একদিন এক গ্রামে..." }
  4. API key থাকলে header-এ দিন: Authorization: Bearer [API_KEY]
  5. Fallback, memory, context—all supported

ডিটেইল API spec: docs/api_spec.yaml
