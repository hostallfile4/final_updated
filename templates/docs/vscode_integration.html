VSCode Integration Guide

Learn how to integrate and use our AI agents in Visual Studio Code

Quick Start Guide

Setup Instructions

1

Install Extension

Install our VSCode extension from the marketplace:

ext install zombie-coder.ai-assistant
2

Configure API Endpoint

Set your API endpoint in VSCode settings:

  1. Open Command Palette (Ctrl+Shift+P)
  2. Search for "ZombieCoder: Set API Endpoint"
  3. Enter your endpoint URL
3

Test Connection

Verify the connection:

  1. Open Command Palette
  2. Run "ZombieCoder: Test Connection"
  3. You should see a success message
