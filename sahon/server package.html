২. প্রয়োজনীয় প্যাকেজসমূহ

২.১ ওয়েব ফ্রেমওয়ার্ক

  • flask==3.1.1 - ওয়েব অ্যাপ্লিকেশন ফ্রেমওয়ার্ক
  • fastapi==0.116.1 - আধুনিক, দ্রুত ওয়েব ফ্রেমওয়ার্ক
  • uvicorn==0.35.0 - ASGI সার্ভার

২.২ অডিও প্রসেসিং

  • pyttsx3==2.99 - টেক্সট-টু-স্পীচ ইঞ্জিন
  • gTTS==2.5.4 - Google টেক্সট-টু-স্পীচ API

২.৩ সিস্টেম টুলস

  • psutil==7.0.0 - সিস্টেম এবং প্রসেস মনিটরিং
  • python-dotenv==1.1.1 - এনভায়রনমেন্ট ভ্যারিয়েবল ম্যানেজমেন্ট
  • watchdog==6.0.0 - ফাইল সিস্টেম মনিটরিং

২.৪ অন্যান্য টুলস

  • markdown2==2.5.3 - মার্কডাউন প্রসেসিং
  • flask-cors==6.0.1 - ক্রস-অরিজিন রিসোর্স শেয়ারিং
  • PyYAML==6.0.2 - YAML ফাইল প্রসেসিং

৩. ইনস্টলেশন নির্দেশনা

৩.১ ভার্চুয়াল এনভায়রনমেন্ট সেটআপ

# ভার্চুয়াল এনভায়রনমেন্ট তৈরি python -m venv .venv # ভার্চুয়াল এনভায়রনমেন্ট অ্যাক্টিভেট .\.venv\Scripts\activate

৩.২ প্যাকেজ ইনস্টলেশন

# pip আপডেট python -m pip install -U pip # requirements.txt থেকে ইনস্টল python -m pip install -r requirements.txt

৩.৩ সার্ভার স্টার্টআপ ক্রম

# ১. MCP সার্ভার python mcp_server.py # ২. মডেল সার্ভার python app.py # ৩. মেইন সার্ভার cd api && python app.py # ৪. ক্লায়েন্ট API python client_monitoring_api.py # ৫. অডিও API python zonemind_audio_api.py

বিশেষ দ্রষ্টব্য:

  • প্রতিটি সার্ভার আলাদা টার্মিনাল উইন্ডোতে চালাতে হবে
  • সার্ভারগুলো অবশ্যই উপরের ক্রম অনুযায়ী চালাতে হবে
  • প্রতিটি সার্ভার চালু করার পর স্টেটাস চেক করতে হবে
  • কোন সার্ভার চালু না হলে লগ ফাইল চেক করতে হবে
